### Node.js UDP Chat example

This is just a mundane UDP chat using node.js for learning purpose.

## Running server

```bash
node server.js
```
## Running client

```bash
node client.js
```

## Note

Test it out if you don't know Node.js or UDP protocol.
Feel free to "contribute" :)
